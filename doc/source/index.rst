kokex API
=================================


한국어 문서 집합에서 키워드를 추출하는 라이브러리입니다.
구문분석을 통해 사전 없이도 고유명사, 복합명사, 신조어에 강건한 키워드 추출기를 만들고자 합니다.

이 문서는 kokex 내부에 대한 지식 없이도 API 를 사용할 수 있도록,
각 API의 입력 및 리턴 값을 설명할 예정입니다.
이런 경우에는, kokex.api 모듈의 함수만 빠르게 확인하시면 충분합니다.
하지만, 그 외 다른 내부 개발 모듈은 docstring 적용이 미흡한 상태이며 추후 개선이 필요합니다.

소스코드는 `Github <https://github.com/jsmyung-datansoft/kokex>`_ 에서 확인 가능하기 때문에, 필요한 부분은 소스코드를 참고해주시면 좋겠습니다.
